The Sales Management module provides a flexible system for recording product sales and service transactions. It integrates with the inventory system to track stock levels and supports both registered clients and manual customer entries.

## Overview

The sales module (`public/pages/admin/ventas.html`) enables administrators to:

- Record product-based sales with inventory integration
- Register service-only transactions
- Track sales by client
- Support manual customer entry for walk-in sales
- Generate sales reports and analytics
- Export sales data to Excel and PDF

## Sales Workflow

### Recording a Product Sale

<Steps>
  <Step title="Open sales form">
    Click **Registrar venta** button to open the sales registration modal.
  </Step>
  
  <Step title="Select client">
    Choose a registered client from the dropdown or enable manual entry for walk-in customers.
    
    ```javascript ventas.js
    // Client selection modes
    const chkClienteManual = document.getElementById("chkClienteManual");
    const colClienteSelect = document.getElementById("colClienteSelect");
    const colClienteInput = document.getElementById("colClienteInput");
    ```
  </Step>
  
  <Step title="Add products">
    - Select product category
    - Choose specific product
    - Enter quantity
    - View unit price and total automatically calculated
    - Click **Agregar** to add to cart
  </Step>
  
  <Step title="Review and save">
    - Review all items in the cart
    - Verify total amount
    - Click **Guardar Venta** to complete the transaction
  </Step>
</Steps>

### Recording a Service Transaction

<Steps>
  <Step title="Enable service mode">
    Check the **Servicio** toggle to switch to service-only mode.
    
    ```javascript ventas.js lines
    // Service mode listener
    if (chkServicio) {
        chkServicio.addEventListener("change", () => {
            if (chkServicio.checked) {
                colCategoria.style.display = "none";
                colProducto.style.display = "none";
                colServicioDesc.style.display = "block";
            } else {
                colCategoria.style.display = "block";
                colProducto.style.display = "block";
                colServicioDesc.style.display = "none";
            }
        });
    }
    ```
  </Step>
  
  <Step title="Enter service details">
    - Describe the service provided
    - Enter the service amount manually
  </Step>
  
  <Step title="Complete transaction">
    Save the service transaction without affecting inventory levels.
  </Step>
</Steps>

## Data Structure

### Sale Document

Sales are stored in the `ventas` Firestore collection:

```typescript
interface Venta {
  id: string;
  fecha: Timestamp;
  clienteId?: string;          // If registered client
  clienteNombre: string;        // Client name
  tipoVenta: "producto" | "servicio";
  items: VentaItem[];
  total: number;
  metodoPago?: string;
  estado: "completada" | "pendiente" | "cancelada";
  registradoPor: string;        // Admin user ID
  notas?: string;
  createdAt: Timestamp;
  updatedAt: Timestamp;
}

interface VentaItem {
  tipo: "producto" | "servicio";
  nombre: string;
  cantidad: number;
  precioUnitario: number;
  subtotal: number;
  // For products
  productoId?: string;
  categoriaId?: string;
  // For services
  descripcionServicio?: string;
}
```

## Sales Features

### Client Database Integration

The sales module integrates with the `usuario` collection to access registered clients:

```javascript ventas.js
const selectClientes = document.getElementById("clienteSel");

// Load registered clients
async function cargarClientes() {
  const querySnapshot = await getDocs(
    query(collection(db, "usuario"), where("rol", "==", "Cliente"))
  );
  
  querySnapshot.forEach((doc) => {
    const data = doc.data();
    const option = document.createElement("option");
    option.value = doc.id;
    option.textContent = `${data.nombre} ${data.apellido}`;
    selectClientes.appendChild(option);
  });
}
```

### Stock Management

When recording product sales, the system automatically updates inventory:

```javascript ventas.js
// Update stock after sale
await runTransaction(db, async (transaction) => {
  const productoRef = doc(db, "articulo", productoId);
  const productoDoc = await transaction.get(productoRef);
  
  if (productoDoc.exists()) {
    const stockActual = productoDoc.data().stock;
    transaction.update(productoRef, {
      stock: stockActual - cantidad,
      ultimaVenta: serverTimestamp()
    });
  }
});
```
