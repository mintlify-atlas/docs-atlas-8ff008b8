## Overview

The Reports (Reportes) module provides comprehensive analytics and reporting capabilities across all TextileFlow ERP modules, with support for attendance tracking, payroll analysis, and customizable report generation.

<Note>
  Reports support multiple export formats (PDF, Excel, Print) and include interactive charts using Chart.js.
</Note>

**Report Calculation** (`reportes.js:436-458`):

```javascript
Object.entries(porEmpleadoFecha).forEach(([userId, fechas]) => {
  Object.values(fechas).forEach(dia => {
    if (dia.entrada) {
      usuariosMap[userId].totalDias++;

      if (dia.salida) {
        usuariosMap[userId].diasCompletos++;
        
        // Calculate hours worked
        const [hE, mE] = dia.entrada.split(':').map(Number);
        const [hS, mS] = dia.salida.split(':').map(Number);
        const minutosTotal = (hS * 60 + mS) - (hE * 60 + mE);
        usuariosMap[userId].totalHoras += minutosTotal / 60;
      }

      // Detect tardiness (after 8:30 AM)
      const [h, m] = dia.entrada.split(':').map(Number);
      if (h > 8 || (h === 8 && m > 30)) {
        usuariosMap[userId].tardanzas++;
      }
    }
  });
});
```

**Attendance Percentage**:
- ≥ 90% = Green badge (status-complete)
- 70-89% = Yellow badge (status-incomplete)
- < 70% = Red badge (status-absent)

## Export Functionality

### Excel Export

The `exportToExcel()` function (`reportes.js:619-772`) supports all report types:

```javascript
window.exportToExcel = function () {
  if (!currentReportData) {
    alert('No hay datos para exportar');
    return;
  }

  const wb = XLSX.utils.book_new();
  let data = [];
  let sheetName = 'Reporte';

  // Prepare data based on report type
  if (currentReportData.type === 'asistencia-mensual') {
    sheetName = 'Resumen Mensual';
    data.push(['Empleado', 'Total Días', 'Días Completos', 'Tardanzas', ...]);
    Object.values(currentReportData.data).forEach(stats => {
      data.push([stats.displayName, stats.totalDias, ...]);
    });
  } else if (currentReportData.type === 'tardanzas') {
    sheetName = 'Tardanzas';
    data.push(['Empleado', 'Fecha', 'Hora Entrada', 'Minutos Tarde']);
    currentReportData.data.forEach(t => {
      data.push([t.displayName, t.fecha, t.hora, t.minutosTarde]);
    });
  } else if (currentReportData.type === 'nomina-mensual') {
    // Payroll export with totals
  }

  const ws = XLSX.utils.aoa_to_sheet(data);
  XLSX.utils.book_append_sheet(wb, ws, sheetName);
  XLSX.writeFile(wb, `reporte_${currentReportData.type}_${currentReportData.month}.xlsx`);
};
```

**Excel Features**:
- Automatic column headers
- Summary totals for payroll reports
- Descriptive sheet names
- Date-stamped filenames

```typescript
interface CurrentReportData {
  type: 'asistencia-mensual' | 'tardanzas' | 'nomina-mensual' | 'por-empleado' | 'comparativo';
  month?: string;  // "YYYY-MM"
  data: any;  // Report-specific data structure
  totales?: {
    salarios?: number;
    bonos?: number;
    deducciones?: number;
    neto?: number;
    cantidad?: number;
  };
  meses?: string[];  // For comparative reports
  empleado?: string;  // For employee-specific reports
}
```

## Helper Functions

### Month Name Formatting

```javascript
function formatMonthName(mesYao) {
  const [year, month] = mesYao.split('-').map(Number);
  const meses = ['Enero', 'Febrero', 'Marzo', 'Abril', 'Mayo', 'Junio',
                 'Julio', 'Agosto', 'Septiembre', 'Octubre', 'Noviembre', 'Diciembre'];
  return `${meses[month - 1]} ${year}`;
}
// Input: "2026-03" → Output: "Marzo 2026"
```

### Loading Employee Filter

```javascript
async function loadEmployeeFilter() {
  const select = document.getElementById('filterEmployee');
  const snapshot = await getDocs(collection(db, "usuario"));
  
  let optionsHtml = '<option value="">Todos los empleados</option>';
  snapshot.forEach(doc => {
    const user = doc.data();
    const displayName = user.displayName || `${user.nombre} ${user.apellido}` || user.email;
    optionsHtml += `<option value="${doc.id}">${displayName}</option>`;
  });
  
  select.innerHTML = optionsHtml;
}
```
