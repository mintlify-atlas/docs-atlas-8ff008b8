## Authentication & Security

### Firebase Authentication Integration

TextileFlow uses Firebase Authentication v10.13.1 for secure user management:

```javascript
import { 
  signInWithEmailAndPassword,
  createUserWithEmailAndPassword,
  sendPasswordResetEmail 
} from "https://www.gstatic.com/firebasejs/10.13.1/firebase-auth.js";

const userCredential = await signInWithEmailAndPassword(auth, email, password);
const user = userCredential.user;
```

<Note>
  The system validates user credentials against Firebase Auth and retrieves user data from Firestore to determine role-based access.
</Note>

### Role-Based Access Control (RBAC)

Two primary roles with distinct permissions:

- **Admin**: Full system access including RRHH, inventory, attendance, payroll, reports, and configuration
- **Empleado** (Employee): Limited access to personal attendance and profile management

Session data is stored in localStorage with the following structure:

```javascript
{
  uid: string,
  email: string,
  nombre: string,
  apellido: string,
  displayName: string,
  rol: "admin" | "Empleado",
  timestamp: number
}
```

### Route Protection

The AuthGuard component protects routes based on user authentication and role:

```javascript
const session = localStorage.getItem('textileflow_session');
if (session) {
  const sessionData = JSON.parse(session);
  if (sessionData.rol === 'admin' || sessionData.rol === 'Administrador') {
    window.location.replace('./admin/dashboard.html');
  } else {
    window.location.replace('./employee/dashboard.html');
  }
}
```

### Advanced Search & Filtering

Real-time search implementation with multiple filters:

```javascript
const filteredUsers = allUsers.filter(cliente => {
  // Search by document, name, or email
  const matchesSearch = 
    cliente.documento?.toLowerCase().includes(searchTerm) ||
    `${cliente.nombre} ${cliente.apellido}`.toLowerCase().includes(searchTerm) ||
    cliente.email?.toLowerCase().includes(searchTerm);
  
  // Filter by document type, role, and status
  const matchesTipoDoc = !filterTipoDoc || cliente.tipoDocumento === filterTipoDoc;
  const matchesRol = !filterRol || cliente.rol === filterRol;
  const matchesEstado = !filterEstado || cliente.estado === filterEstado;
  
  return matchesSearch && matchesTipoDoc && matchesRol && matchesEstado;
});
```

### Data Validation

Uniqueness validation for documents and emails:

```javascript
async function checkDocumentExists(documento, excludeUid = null) {
  const q = query(collection(db, "usuario"), where("documento", "==", documento));
  const querySnapshot = await getDocs(q);
  
  if (excludeUid) {
    return querySnapshot.docs.some(doc => doc.id !== excludeUid);
  }
  
  return !querySnapshot.empty;
}
```

<Warning>
  Always validate document and email uniqueness before creating or updating employee records to prevent data conflicts.
</Warning>

### Loading Attributes for Articles

```javascript
async function cargarAtributosEnSelect() {
  selectAtributos.innerHTML = `<option value="">Seleccione un atributo</option>`;
  
  const querySnapshot = await getDocs(collection(db, "atributo"));
  querySnapshot.forEach((doc) => {
    const data = doc.data();
    const option = document.createElement("option");
    option.value = data.id_atributo;
    option.textContent = data.nombre_atributo;
    selectAtributos.appendChild(option);
  });
}
```

## Attendance System

### Time Zone Configuration

All attendance records use Lima, Peru timezone:

```javascript
function getTodayLima() {
  const now = new Date();
  const limaTime = new Date(now.toLocaleString('en-US', { 
    timeZone: 'America/Lima' 
  }));
  return limaTime.toISOString().split('T')[0];
}

function getCurrentTimeLima() {
  const now = new Date();
  const limaTime = new Date(now.toLocaleString('en-US', { 
    timeZone: 'America/Lima' 
  }));
  const hours = limaTime.getHours().toString().padStart(2, '0');
  const minutes = limaTime.getMinutes().toString().padStart(2, '0');
  return `${hours}:${minutes}`;
}
```

### Configurable Schedules

Default and custom schedules per employee:

```javascript
let defaultSchedule = { 
  horaEntrada: "08:00", 
  horaSalida: "17:00", 
  tolerancia: 15 // minutes
};

async function loadDefaultSchedule() {
  const docRef = doc(db, "configuracion", "horario_default");
  const docSnap = await getDoc(docRef);
  
  if (docSnap.exists()) {
    defaultSchedule = docSnap.data();
  } else {
    await setDoc(docRef, defaultSchedule);
  }
}
```
